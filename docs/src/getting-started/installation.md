# Installation

Get `reflectapi` up and running in minutes.

## Basic Setup

```shell
cargo add reflectapi --features builder,axum,json,chrono
```



## CLI Tool

Install the CLI tool to generate client libraries:

```bash
cargo install reflectapi-cli
```
